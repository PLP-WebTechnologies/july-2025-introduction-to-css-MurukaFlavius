About CSS

CSS is used to style and format HTML documents. It controls layout, colors, fonts, spacing, and more.

The Box Model

Every HTML element is a box made of four parts:

  • Content — the actual text or image
  • Padding — space between content and border
  • Border — wraps around the padding
  • Margin — space outside the border
